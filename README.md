# GSTR-2B to Tally Import Converter

A high-speed, professional utility designed for accountants and tax professionals performing **emergency/last-minute accounting before ITR filing**. This tool is specifically built to help you finalize books of accounts by pulling accurate data directly from the GST Portal and converting it into Tally-ready entries in seconds.

## 🕒 The "Last-Minute" Solution
When ITR deadlines are approaching and manual data entry is not an option, this tool allows you to:
- **Instant Bookkeeping:** Record an entire year's worth of purchases by processing monthly GSTR-2B files.
- **Ensure 100% Accuracy:** Eliminate manual typing errors; the tool uses the exact values (Taxable, IGST, CGST, SGST) reported by suppliers.
- **Match 26AS/AIS:** By accounting based on GSTR-2B, your purchase data will align perfectly with the GST credits reflected in your Income Tax portals.

## ✨ Key Features
- **Voucher Automation:** Automatically generates Multi-line Journals/Purchases (Cr. Supplier, Dr. Purchase, Dr. Taxes).
- **Custom Ledger Mapping:** Instantly map the data to your specific Tally ledger names (e.g., "Input SGST @ 9%").
- **Zero Installation:** Works offline and locally in your browser. No server uploads means total data privacy during sensitive tax audits.
- **Smart Date Correction:** Converts all portal dates to the Tally-standard **DD-MM-YYYY** format to prevent import errors.

## 🚀 Speed Run: How to use it
1. **Open** `index.html` in your browser.
2. **Set Ledgers:** Enter your Tally ledger names once in the configuration panel.
3. **Upload:** Drop your GSTR-2B Excel file (downloaded from the GST portal).
4. **Convert:** Hit download to get the "Tally Import" Excel file.
5. **Finalize:** In Tally, go to **Import Data > Vouchers** and select the file. Your books are now updated!

## 🛠 Technical Details
- **Privacy:** 100% Client-side. Your data never leaves your PC.
- **Compatibility:** Optimized for Tally ERP 9 and TallyPrime.
- **Library:** Powered by [SheetJS](https://github.com/SheetJS/sheetjs).
